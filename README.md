# The newvbtm Package
The `newvbtm` package defines general purpose macro named `\newverbatim` to define your own verbatim-like environment. It also has a supplementary style file `varvbtm.sty` to provide set of macros for variants of verbatim, such as tab emulation.

## Note
This GitHub repository should only be used for reporting issues; the code is developed privately off GitHub on my local computer.

The official versions of the bundle are released on [CTAN](https://ctan.org/pkg/newvbtm).
